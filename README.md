
# TokukoPreferences

A lightweight World of Warcraft addon that automatically performs the **/oom** emote when the player exits combat **and** their mana is below a configurable percentage.

**Features**
- Enable/Disable toggle
- Mana threshold slider (1–100%)
- “Only in Group/Instance” toggle (avoid solo emotes)
- Modern Settings Panel (Dragonflight + TWW)

## Installation
1. Extract the folder **TokukoPreferences** into:
   ```
   World of Warcraft/_retail_/Interface/AddOns/
   ```
2. Start the game or `/reload`.
3. Configure under **Esc → Options → AddOns → TokukoPreferences**.

## Versioning
The `.toc` has `## Version: @project-version@` so you can replace that token at build time (e.g., from a Git tag).
